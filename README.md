# Лабораторные работы — криптосистемы Осипяна

Реализация шести лабораторных работ по курсу «Системы передачи и защиты информации» на основе модели алфавитной криптосистемы В.О. Осипяна:

> **Σ₀ = (M\*, Q, C\*, E(m), D(c) | V(E(m), D(c)))**

## Структура проекта

```
osipyan_labs/
├── main.py                         # CLI-приложение (точка входа)
├── REPORT.md                       # Общий отчёт о проделанной работе
├── README.md                       # Этот файл
├── requirements.txt                # Зависимости (PyQt6)
├── task_1/
│   ├── hamming_code.py             # Код Хэмминга C(n, k)
│   └── REPORT.md                   # Отчёт по задаче 1
├── task_2/
│   ├── hamming_extended.py         # Расширенный код Хэмминга C[n+1, k]
│   └── REPORT.md                   # Отчёт по задаче 2
├── task_3/
│   ├── varshamov_code.py           # Код Варшамова-Тененгольца VT_a(n)
│   └── REPORT.md                   # Отчёт по задаче 3
├── task_4/
│   ├── rsa_crypto.py               # RSA (GUI + CLI)
│   └── REPORT.md                   # Отчёт по задаче 4
├── task_5/
│   ├── knapsack_crypto.py          # Аддитивный рюкзак АВКР (GUI + CLI)
│   └── REPORT.md                   # Отчёт по задаче 5
└── task_6/
    ├── multiplicative_knapsack_crypto.py  # Мультипликативный рюкзак МВКР (GUI + CLI)
    └── REPORT.md                   # Отчёт по задаче 6
```

## Список задач

| # | Задача | Тип | Режим |
|---|--------|-----|-------|
| 1 | Код Хэмминга C(n, k) — исправление 1 симметричной ошибки | Помехоустойчивое кодирование | CLI |
| 2 | Расширенный код Хэмминга C[n+1, k] — исправление 1, обнаружение 2 ошибок | Помехоустойчивое кодирование | CLI |
| 3 | Код Варшамова-Тененгольца VT_a(n) — исправление 1 асимметричной ошибки | Помехоустойчивое кодирование | CLI |
| 4 | RSA — криптосистема (NP: факторизация) | Криптография с открытым ключом | GUI (PyQt6) |
| 5 | Аддитивный рюкзак АВКР (NP: Subset Sum) | Криптография с открытым ключом | GUI (PyQt6) |
| 6 | Мультипликативный рюкзак МВКР (NP: Subset Product) | Криптография с открытым ключом | GUI (PyQt6) |

## Установка

```bash
# Клонируйте или скопируйте проект
cd osipyan_labs

# Установите зависимости
pip install -r requirements.txt
```

**Требования:** Python 3.12+, PyQt6 (для задач 4-6 с GUI).

## Запуск

### Интерактивное меню (рекомендуется)

```bash
python main.py
```

Откроется главное меню с выбором задачи:
- Задачи 1-3 работают в интерактивном CLI-режиме (кодирование, декодирование, верификация).
- Задачи 4-6 открывают окно GUI (PyQt6).

### Запуск отдельных задач

```bash
# Задачи 1-3 (консольная демонстрация)
python task_1/hamming_code.py
python task_2/hamming_extended.py
python task_3/varshamov_code.py

# Задачи 4-6 (GUI по умолчанию)
python task_4/rsa_crypto.py
python task_5/knapsack_crypto.py
python task_6/multiplicative_knapsack_crypto.py

# Задачи 4-6 (консольный режим)
python task_4/rsa_crypto.py --cli
python task_5/knapsack_crypto.py --cli
python task_6/multiplicative_knapsack_crypto.py --cli
```
