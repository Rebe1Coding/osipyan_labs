# Код Варшамова-Тененгольца $VT_a(n)$ — передача данных с исправлением одной асимметричной ошибки

## 1. Идея метода (схематично)

Код Варшамова-Тененгольца (VT-код) — это нелинейный двоичный код, предназначенный для **обнаружения и исправления одной асимметричной ошибки** при передаче данных по каналу связи.

### Отличие от симметричных кодов

В **симметричном** канале (модель BSC) ошибки равновероятны в обоих направлениях: $0 \to 1$ и $1 \to 0$. Для таких каналов применяется, например, код Хэмминга.

В **асимметричном** канале (Z-канал) ошибки происходят **только в одном направлении**:
- Z-канал: $1 \to 0$ (единица может стать нулём, но не наоборот);
- Обратный Z-канал: $0 \to 1$ (нуль может стать единицей, но не наоборот).

Такая модель естественна для многих физических каналов связи и систем хранения данных, где один тип искажения доминирует.

### Схема передачи данных

```
Отправитель                     Z-канал (асиммет.)            Получатель
┌───────────┐                ┌──────────────────┐          ┌──────────────┐
│ Сообщение │   E(m)         │  Асимметричная   │  D(c')   │ Вычисление   │
│ m (k бит) │ ──────────►    │  ошибка:         │ ───────► │ синдрома     │
│           │  Табличное     │  1→0 (или 0→1)   │          │ S(y) mod(n+1)│
│           │  кодирование   │  в одном бите    │          │              │
│           │  c ∈ VT_a(n)   │                  │          │ S=a → нет    │
│           │  (n бит)       │  c' = c − e_j    │          │   ошибки     │
│           │                │  (один бит       │          │ S≠a → позиция│
│           │                │   обнулился)     │          │   j, исправл.│
└───────────┘                └──────────────────┘          └──────────────┘
                                                                 │
                                                                 ▼
                                                          ┌──────────────┐
                                                          │ Поиск слова  │
                                                          │ в кодовой    │
                                                          │ книге → m    │
                                                          └──────────────┘
```

В терминологии криптосистемы Осипяна задача оформляется в виде кортежа:

$$\Sigma_0 = (M^{\ast}, Q, C^{\ast}, E(m), D(c) \mid V(E(m), D(c)))$$

где:
- $M^{\ast}$ — множество всех открытых текстов (сообщений) длины $k$;
- $Q$ — алфавит $\{0, 1\}$;
- $C^{\ast}$ — множество кодовых слов (шифротекстов) длины $n$, принадлежащих $VT_a(n)$;
- $E(m)$ — алгоритм кодирования (шифрования): $m \to c \in VT_a(n)$;
- $D(c)$ — алгоритм декодирования (дешифрования): $c \to m$;
- $V(E(m), D(c))$ — верификация: $D(E(m)) = m$ для любого $m \in M^{\ast}$.

Алгоритмы $E(m)$ и $D(c)$ связаны тем, что для любого сообщения $m = m_1 m_2 \ldots m_k \in M^{\ast}$ однозначно преобразуется в кодовое слово $c = c_1 c_2 \ldots c_n \in C^{\ast}$, и обратно: по кодовому слову $c$ всегда можно восстановить исходное сообщение $m$.

## 2. Математическое описание

### 2.1. Определение кода Варшамова-Тененгольца

Код Варшамова-Тененгольца $VT_a(n)$ определяется как множество двоичных слов длины $n$, удовлетворяющих условию:

$$VT_a(n) = \left\{ x = (x_1, x_2, \ldots, x_n) \in \{0,1\}^n \;\middle|\; \sum_{i=1}^{n} i \cdot x_i \equiv a \pmod{n+1} \right\}$$

где:
- $n$ — длина кодового слова;
- $a$ — параметр кода, $0 \leq a \leq n$;
- $x_i \in \{0, 1\}$ — $i$-й бит кодового слова;
- сумма $\sum_{i=1}^{n} i \cdot x_i$ называется **синдромом Варшамова**.

### 2.2. Синдром Варшамова

Для произвольного двоичного слова $y = (y_1, y_2, \ldots, y_n)$ синдром определяется как:

$$S(y) = \sum_{i=1}^{n} i \cdot y_i \pmod{n+1}$$

Для любого кодового слова $c \in VT_a(n)$ выполняется $S(c) = a$.

### 2.3. Мощность кода и информационная ёмкость

Мощность кода $|VT_a(n)|$ приближённо равна:

$$|VT_a(n)| \approx \frac{2^n}{n+1}$$

Это следует из того, что все $n+1$ подмножеств $VT_0(n), VT_1(n), \ldots, VT_n(n)$ разбивают $\{0,1\}^n$ на непересекающиеся классы.

Число информационных бит:

$$k = \lfloor \log_2 |VT_a(n)| \rfloor$$

Скорость кода:

$$R = \frac{k}{n} \approx 1 - \frac{\log_2(n+1)}{n}$$

При $n \to \infty$ скорость $R \to 1$, что делает VT-коды **асимптотически оптимальными**.

### 2.4. Кодирование $E(m)$

Кодирование выполняется **табличным методом**:

1. Генерируется множество $C^{\ast} = VT_a(n)$ — все кодовые слова, удовлетворяющие условию синдрома.
2. Кодовые слова упорядочиваются лексикографически и нумеруются: $c_0, c_1, \ldots, c_{|C^{\ast}|-1}$.
3. Сообщение $m = (m_1, m_2, \ldots, m_k)$ интерпретируется как двоичное число $I(m) = \sum_{j=1}^{k} m_j \cdot 2^{k-j}$.
4. Кодовое слово: $c = E(m) = c_{I(m)}$.

### 2.5. Декодирование $D(c)$ с исправлением ошибки

Принято слово $y = (y_1, y_2, \ldots, y_n)$, возможно содержащее одну асимметричную ошибку.

**Шаг 1.** Вычисляем синдром принятого слова:

$$S(y) = \sum_{i=1}^{n} i \cdot y_i \pmod{n+1}$$

**Шаг 2.** Анализ синдрома:

- Если $S(y) = a$, ошибок нет: $y$ — корректное кодовое слово.
- Если $S(y) \neq a$, произошла ошибка. Позиция ошибки определяется в зависимости от типа канала:

**Z-канал (ошибки $1 \to 0$):** если бит $x_j = 1$ стал $y_j = 0$, то синдром уменьшился на $j$:

$$S(y) = S(x) - j \pmod{n+1} \implies j = (a - S(y)) \bmod{(n+1)}$$

**Обратный Z-канал (ошибки $0 \to 1$):** если бит $x_j = 0$ стал $y_j = 1$, то синдром увеличился на $j$:

$$S(y) = S(x) + j \pmod{n+1} \implies j = (S(y) - a) \bmod{(n+1)}$$

**Шаг 3.** Исправление: инвертируем бит на позиции $j$.

**Шаг 4.** Поиск исправленного слова в кодовой книге для извлечения сообщения $m$.

### 2.6. Верификация $V(E(m), D(c))$

Свойство корректности криптосистемы:

$$\forall m \in M^{\ast}: \quad D(E(m)) = m$$

## 3. Реализация (код)

```python
"""
Код Варшамова-Тененгольца (VT-код) — передача данных по каналу связи
с обнаружением и исправлением одной асимметричной ошибки.

Оформлено в соответствии с моделью алфавитной криптосистемы
В.О. Осипяна: Σ₀ = (M*, Q, C*, E(m), D(c) | V(E(m), D(c))).

Автор алгоритма: В.О. Осипян
Литература: Осипян В.О. Элементы теории передачи информации.
            Краснодар, 1998.
"""

from __future__ import annotations

import math
import random
from dataclasses import dataclass, field
from itertools import product


# ---------------------------------------------------------------------------
#  Вспомогательные функции
# ---------------------------------------------------------------------------

def _varshamov_syndrome(word: list[int], n: int) -> int:
    """Вычисляет синдром Варшамова-Тененгольца для двоичного слова.

    Формула: S(x) = Σ_{i=1}^{n} i·x_i  (mod n+1)

    Позиции нумеруются с 1: x_1, x_2, ..., x_n.
    В массиве Python индексация с 0, поэтому i-я позиция → word[i-1].
    """
    # Используем генераторное выражение для компактности.
    # (i + 1) — потому что в формуле нумерация с 1, а в Python с 0.
    # Модуль (n + 1) берётся в конце, а не на каждом шаге —
    # для целых чисел это эквивалентно, но быстрее.
    return sum((i + 1) * word[i] for i in range(n)) % (n + 1)


def _generate_vt_codebook(n: int, a: int) -> list[list[int]]:
    """Генерирует все кодовые слова VT_a(n).

    VT_a(n) = { x ∈ {0,1}^n : Σ_{i=1}^{n} i·x_i ≡ a (mod n+1) }

    Перебираем все 2^n двоичных слов и отбираем те,
    чей синдром равен a. Это прямая реализация определения кода.
    Для практических целей (n ≤ 20) перебор работает быстро.
    """
    codebook: list[list[int]] = []
    for bits in product((0, 1), repeat=n):
        word = list(bits)
        if _varshamov_syndrome(word, n) == a:
            codebook.append(word)
    return codebook


def _bits_to_str(bits: list[int]) -> str:
    """Превращает список бит в строку для удобного вывода."""
    return "".join(map(str, bits))


def _bits_to_int(bits: list[int]) -> int:
    """Преобразует двоичный вектор в целое число."""
    result = 0
    for b in bits:
        result = result * 2 + b
    return result


# ---------------------------------------------------------------------------
#  Класс-криптосистема кода Варшамова-Тененгольца
# ---------------------------------------------------------------------------

@dataclass
class VarshamovCodeSystem:
    """Реализация криптосистемы Σ₀ = (M*, Q, C*, E(m), D(c) | V(E(m), D(c)))
    на основе кода Варшамова-Тененгольца VT_a(n).

    Код VT_a(n) предназначен для исправления **одной асимметричной** ошибки
    в двоичном канале связи (Z-канал). В отличие от кода Хэмминга,
    который исправляет симметричные ошибки (0↔1 с равной вероятностью),
    VT-код ориентирован на каналы, где ошибки однонаправлены:
    преимущественно 1→0 или преимущественно 0→1.

    В терминологии Осипяна
    ----------------------
    M*  — множество всех сообщений (двоичных строк длины k).
    Q   — множество числовых эквивалентов элементарных сообщений {0, 1}.
    C*  — множество кодовых слов (шифротекстов) длины n, принадлежащих VT_a(n).
    E(m) — алгоритм кодирования (шифрования): m → c ∈ VT_a(n).
    D(c) — алгоритм декодирования (дешифрования): c → m.
    V(E(m), D(c)) — верификация: D(E(m)) == m для любого m ∈ M*.
    """

    n: int  # длина кодового слова
    a: int = 0  # параметр кода
    k: int = 0  # число информационных бит (вычисляется)
    _codebook: list[list[int]] = field(
        default_factory=list, repr=False, init=False
    )

    def __post_init__(self) -> None:
        if self.n < 2:
            raise ValueError(f"n должно быть >= 2, получено {self.n}")
        if not (0 <= self.a <= self.n):
            raise ValueError(
                f"Параметр a должен быть в диапазоне [0, {self.n}], "
                f"получено {self.a}"
            )
        # Генерируем кодовую книгу — множество C*.
        # Это центральная структура данных: упорядоченный список
        # всех слов, удовлетворяющих условию синдрома.
        self._codebook = _generate_vt_codebook(self.n, self.a)
        # Число информационных бит = ⌊log₂|C*|⌋.
        # Берём степень двойки, чтобы обеспечить биективное отображение
        # между сообщениями и кодовыми словами.
        self.k = int(math.log2(len(self._codebook)))

    # ---- E(m): кодирование (шифрование) ------------------------------------

    def encode(self, message: list[int]) -> list[int]:
        """Кодирует информационное слово m длины k в кодовое слово c ∈ VT_a(n).

        Это функция E(m) криптосистемы — прямое преобразование.

        Используется табличное кодирование: сообщение m интерпретируется
        как двоичное число — индекс в кодовой книге VT_a(n).
        Такой подход гарантирует, что результат всегда принадлежит VT_a(n),
        т.к. кодовая книга содержит только слова с правильным синдромом.
        """
        self._validate_message(message)

        # Интерпретируем сообщение как номер кодового слова.
        # Это реализация формулы: I(m) = Σ m_j · 2^(k-j).
        index = _bits_to_int(message)

        if index >= len(self._codebook):
            raise ValueError(
                f"Индекс сообщения {index} выходит за пределы "
                f"кодовой книги (размер {len(self._codebook)})"
            )

        return list(self._codebook[index])

    # ---- D(c): декодирование (дешифрование) --------------------------------

    def decode(
        self, received: list[int], channel_type: str = "1->0"
    ) -> tuple[list[int], int | None, str]:
        """Декодирует принятое слово, исправляя одну асимметричную ошибку.

        Это функция D(c) криптосистемы — обратное преобразование.

        В асимметричном канале (Z-канале) ошибки происходят только
        в одном направлении, которое является свойством канала:
        - Z-канал "1->0": бит может измениться только из 1 в 0;
        - Z-канал "0->1": бит может измениться только из 0 в 1.

        Параметр channel_type определяет тип канала — это априорное
        знание о характеристиках канала связи, необходимое для
        однозначного определения позиции ошибки.
        """
        if channel_type not in ("1->0", "0->1"):
            raise ValueError(
                "Тип канала должен быть '1->0' или '0->1'"
            )
        self._validate_codeword(received)

        corrected = list(received)
        # Вычисляем синдром: S(y) = Σ i·y_i mod (n+1).
        # Это ключевая операция — сравнение S(y) с параметром a.
        syndrome = _varshamov_syndrome(received, self.n)

        error_pos: int | None = None
        error_type = "нет ошибки"

        if syndrome != self.a:
            if channel_type == "1->0":
                # Ошибка 1→0 на позиции j уменьшает синдром на j,
                # поэтому j = (a − S) mod (n+1).
                j = (self.a - syndrome) % (self.n + 1)
                expected_bit = 0  # после ошибки 1→0 бит стал 0
                fix_value = 1     # восстанавливаем в 1
                error_type = "1→0"
            else:
                # Ошибка 0→1 на позиции j увеличивает синдром на j,
                # поэтому j = (S − a) mod (n+1).
                j = (syndrome - self.a) % (self.n + 1)
                expected_bit = 1  # после ошибки 0→1 бит стал 1
                fix_value = 0     # восстанавливаем в 0
                error_type = "0→1"

            if 1 <= j <= self.n and received[j - 1] == expected_bit:
                corrected[j - 1] = fix_value
                error_pos = j - 1  # переводим в 0-based индексацию
            else:
                raise ValueError(
                    f"Не удалось исправить ошибку. "
                    f"Синдром={syndrome}, ожидаемый={self.a}, "
                    f"позиция j={j}. "
                    f"Возможно, произошло более одной ошибки."
                )

        # Поиск исправленного слова в кодовой книге.
        # Индекс слова в книге — это двоичное представление сообщения.
        try:
            index = self._codebook.index(corrected)
        except ValueError:
            raise ValueError(
                "Исправленное слово не найдено в кодовой книге. "
                "Возможно, произошло более одной ошибки."
            )

        # Обратное преобразование: индекс → двоичный вектор длины k.
        decoded = []
        for i in range(self.k - 1, -1, -1):
            decoded.append((index >> i) & 1)

        return decoded, error_pos, error_type

    # ---- V(E(m), D(c)): верификация ----------------------------------------

    def verify(self, message: list[int], channel_type: str = "1->0") -> bool:
        """Проверяет свойство криптосистемы: D(E(m)) == m для данного m.

        Это функция V(E(m), D(c)) — верификация корректности.
        """
        self._validate_message(message)
        encoded = self.encode(message)
        decoded, _, _ = self.decode(encoded, channel_type)
        return decoded == message

    # ---- Имитация канала с асимметричной ошибкой ---------------------------

    def introduce_error(
        self, codeword: list[int], error_type: str = "1->0"
    ) -> tuple[list[int], int | None]:
        """Вносит одну асимметричную ошибку в кодовое слово.

        Моделирует поведение Z-канала: случайным образом выбирает
        бит подходящего значения и инвертирует его.
        """
        received = list(codeword)

        if error_type == "1->0":
            candidates = [i for i, b in enumerate(received) if b == 1]
        elif error_type == "0->1":
            candidates = [i for i, b in enumerate(received) if b == 0]
        else:
            raise ValueError("Тип ошибки должен быть '1->0' или '0->1'")

        if not candidates:
            return received, None

        pos = random.choice(candidates)
        received[pos] ^= 1
        return received, pos

    # ---- Валидация ---------------------------------------------------------

    def _validate_message(self, message: list[int]) -> None:
        if len(message) != self.k:
            raise ValueError(
                f"Длина сообщения должна быть {self.k}, "
                f"получено {len(message)}"
            )
        if not all(b in (0, 1) for b in message):
            raise ValueError("Сообщение должно содержать только 0 и 1")

    def _validate_codeword(self, codeword: list[int]) -> None:
        if len(codeword) != self.n:
            raise ValueError(
                f"Длина кодового слова должна быть {self.n}, "
                f"получено {len(codeword)}"
            )
        if not all(b in (0, 1) for b in codeword):
            raise ValueError("Кодовое слово должно содержать только 0 и 1")
```

|---|---|---|

| $VT_a(n) = \{ x \mid S(x) \equiv a \}$ | `if _varshamov_syndrome(word, n) == a: codebook.append(word)` | Генерация кодовой книги: перебираем все $2^n$ слов и отбираем те, чей синдром равен $a$. Это прямая реализация определения кода. |
| $k = \lfloor \log_2 |C^{\ast}| \rfloor$ | `self.k = int(math.log2(len(self._codebook)))` | Число информационных бит определяется как логарифм мощности кода. `int()` обеспечивает взятие целой части (floor). |
| $I(m) = \sum_{j=1}^{k} m_j \cdot 2^{k-j}$ | `index = _bits_to_int(message)` | Сообщение интерпретируется как двоичное число — индекс в кодовой книге. Функция `_bits_to_int` реализует эту формулу итеративно. |
| $c = E(m) = c_{I(m)}$ | `return list(self._codebook[index])` | Кодирование: выбираем кодовое слово из книги по индексу. Гарантия $c \in VT_a(n)$, т.к. книга содержит только допустимые слова. |
| Z-канал: $j = (a - S(y)) \bmod{(n+1)}$ | `j = (self.a - syndrome) % (self.n + 1)` | Определение позиции ошибки $1 \to 0$. Ошибка уменьшает синдром на $j$, поэтому $j$ восстанавливается из разности. |
| Обр. Z-канал: $j = (S(y) - a) \bmod{(n+1)}$ | `j = (syndrome - self.a) % (self.n + 1)` | Определение позиции ошибки $0 \to 1$. Ошибка увеличивает синдром на $j$. |
| Исправление: $y_j := 1 - y_j$ | `corrected[j - 1] = fix_value` | Инвертируем ошибочный бит. `j - 1` — переход от 1-нумерации к 0-нумерации Python. |
| $D(E(m)) = m$ | `decoded == message` в `verify()` | Прямая проверка свойства верификации $V(E(m), D(c))$ криптосистемы. |

## 5. Пример работы

### Код $VT_0(7)$: $n = 7$, $a = 0$, $k = 4$

**Кодовая книга:** $|VT_0(7)| = 16$ слов, из которых используются все $2^4 = 16$.

Первые кодовые слова:

| Индекс | Кодовое слово | Синдром |
|--------|---------------|---------|
| 0 | `0000000` | 0 |
| 1 | `0001101` | 0 |
| 2 | `0010011` | 0 |
| 3 | `0010100` | 0 |
| 4 | `0100010` | 0 |
| 5 | `0101111` | 0 |

**Кодирование** сообщения $m = (1, 0, 1, 1)$:

$$I(m) = 1 \cdot 2^3 + 0 \cdot 2^2 + 1 \cdot 2^1 + 1 \cdot 2^0 = 11$$

$$c = E(m) = c_{11} = (1, 0, 1, 1, 0, 0, 0)$$

Проверка: $S(c) = 1 \cdot 1 + 0 \cdot 2 + 1 \cdot 3 + 1 \cdot 4 + 0 \cdot 5 + 0 \cdot 6 + 0 \cdot 7 = 8 \equiv 0 \pmod{8}$ ✓

**Декодирование** с ошибкой $1 \to 0$ на позиции $j = 4$ (0-based: 3):

Принято: $y = (1, 0, 1, \mathbf{0}, 0, 0, 0)$

$$S(y) = 1 \cdot 1 + 0 \cdot 2 + 1 \cdot 3 + 0 \cdot 4 + 0 \cdot 5 + 0 \cdot 6 + 0 \cdot 7 = 4$$

$$j = (0 - 4) \bmod{8} = 4$$

Позиция $j = 4$ (0-based: 3), бит $y_4 = 0$ → исправляем на 1.

Исправленное слово: $(1, 0, 1, 1, 0, 0, 0)$ → $m = (1, 0, 1, 1)$ ✓
